# Real Estate & Airbnb Investment Trends Analysis

## Project Overview
A comprehensive data analysis of short-term rental performance across Big Bear and Joshua Tree, 
examining how property characteristics, amenities, and seasonality influence revenue and occupancy rates.

## Objective
Provide data-driven insights to help real estate investors make informed decisions about 
Airbnb property investments through statistical analysis and visualization.

## Technologies & Libraries
- **Python 3** (Pandas, NumPy, Matplotlib, Seaborn)
- **Statistical Analysis** (SciPy t-tests)
- **Data Visualization** (Correlation heatmaps, time-series plots, boxplots)

## Key Findings
**Seasonality Patterns:** Big Bear peaks in winter (ski season); Joshua Tree peaks in spring/fall (hiking season)
**Pool Impact:** Listings with pools show significantly higher average revenue
**Hot Tub Impact:** Listings with hot tubs demonstrate higher occupancy rates
**ROI Insights:** Amenities are measurable drivers of booking demand and revenue generation

## Dataset
- Airbnb market data (2020-2021)
- Property amenities (pools, hot tubs)
- Geolocation data (latitude, longitude)
- Real estate sales data

## Analysis Steps
1. **Data Loading & Cleaning** — Handled semicolon-delimited CSVs with malformed rows
2. **Data Merging** — Unified Airbnb market, amenities, and geolocation datasets
3. **Exploratory Data Analysis** — Distribution analysis, correlation heatmaps
4. **Seasonality Analysis** — Monthly trends by city
5. **Amenity Impact Analysis** — Boxplots and statistical significance testing
6. **Investment Insights** — ROI recommendations based on data patterns

## How to Run
```bash
python real_estate_and_airbnb_investment_trends.py
```
